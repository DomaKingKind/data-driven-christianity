# Data-Driven Christianity

**What the Bible emphasizes when nobody tells it what to say.**

A computational theology project that measures what the biblical text actually talks about using morphological data, Strong's numbers, and embedding similarity ‚ÄĒ with no doctrinal input.

## The Core Finding

When you measure all 13 major theological concepts by frequency, cross-testament continuity, and semantic centrality:

| Rank | Concept | Combined Score |
|------|---------|----------------|
| #1 | Union / Participation / "In God" | Highest |
| #2 | God's Reign / Cosmic Kingship | |
| #3 | Loyal Love / Covenant Faithfulness | |
| ... | ... | |
| #12 | Individual Decision / Personal Salvation | Near bottom |

These are not opinions. They are word counts from Strong's numbers in the STEP Bible morphological corpus.

## How It Works

Each document is generated by a two-step pipeline:

1. **Probe script** (Python) ‚ÄĒ queries `bible.db` and outputs a JSON file
2. **Build script** (Node.js) ‚ÄĒ reads that JSON and generates a formatted `.docx`

```
probe_data_driven_theology.py  ‚Üí  output/data_driven_theology.json
probe_six_extensions.py         ‚Üí output/six_extensions_probe.json
                                         ‚Üď    
                build_data_driven_christianity.js  ‚Üí  Data_Driven_Christianity.docx
```

## Quick Start

```bash
# 1. Install dependencies
pip install numpy requests tqdm
npm install docx

# 2. Build the database from free sources (or download pre-built)
python build_database.py

# 3. Run probes
python probe_data_driven_theology.py
python probe_six_extensions.py
python probe_jesus_paul_preaching.py
python probe_actual_sermons.py
python probe_1cor15_gospel.py
python probe_jesus_gospel.py

# 4. Generate documents
node build_data_driven_christianity.js
node build_jesus_paul_preaching.js
node build_actual_sermons.js
node build_jesus_gospel.js
node build_1cor15_gospel.js
```

## The Database

**bible.db** (685 MB) is a SQLite database containing:

- **282,423** Hebrew OT words with OSHM morphology (STEPBible TAHOT, CC BY 4.0)
- **141,720** Greek NT words with Robinson morphology (STEPBible TAGNT, CC BY 4.0)
- **31,086** BSB verse texts (Public Domain)
- **344,000+** cross-references (OpenBible.info, CC BY)
- **31,086** verse embeddings (BAAI/bge-large-en-v1.5, 1024 dimensions)

You can build it from scratch with `python build_database.py` ‚ÄĒ it downloads everything from free, public sources.
